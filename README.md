# WARNING

Facebook の仕様変更につきちょっと全面的に作り直さないと使えない状態です

# これなに

ひなビタ♪ Facebook ページの投稿を蒐集するやつ

# 環境

Linux 前提

# 使い方

```
$ ./startup.sh
$ ./run.sh
```

これで，`hinabitter.txt` というテキストファイルが得られます。
`hinabitter.txt` は青空文庫形式のテキストです。
また，このレポジトリの `photos/` ディレクトリに取得した画像が配置されます。

取得した投稿データは `raw_data/` に保存されます。また，次回以降はこの取得データは差分更新されます。
